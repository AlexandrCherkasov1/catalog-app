# Охотничий каталог

SPA/SSR-приложение на Next.js App Router для каталога товаров, избранного и корзины.

## Запуск

```bash
npm install
npm run dev
```

Приложение будет доступно на [http://localhost:3000](http://localhost:3000).

## Скрипты

```bash
npm run dev
npm run build
npm run start
npm run lint
npm run typecheck
npm run format
npm run format:check
npm run test
```

## Стек

- Next.js App Router
- TypeScript в strict-режиме
- SCSS Modules
- ESLint
- Prettier
- Husky

Остальные части стека будут добавляться по плану отдельными коммитами.
